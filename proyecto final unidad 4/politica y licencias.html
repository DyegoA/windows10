Politica y privacidad

Avisos y terminos de licencias aplicales

Información del Dispositivo: Windows 10 recopila información sobre el dispositivo que estás utilizando, como el modelo, el sistema operativo, y la configuración. Datos de Uso: Microsoft recopila datos sobre cómo usas Windows 10, incluyendo aplicaciones y servicios que utilizas, configuraciones del sistema y errores. Datos de Diagnóstico: Para mejorar el sistema operativo, Microsoft recopila datos de diagnóstico sobre el rendimiento del sistema y cualquier problema que pueda surgir. Uso de Datos:

Mejora del Servicio: Los datos recopilados ayudan a Microsoft a mejorar la calidad y el rendimiento de Windows 10. Seguridad: Los datos de diagnóstico también ayudan a mejorar la seguridad del sistema operativo. Personalización: La información de uso puede ser utilizada para ofrecerte recomendaciones y experiencias más personalizadas en el sistema. Control de Privacidad:

Configuraciones de Privacidad: Windows 10 permite a los usuarios ajustar sus configuraciones de privacidad. Puedes controlar qué datos se recopilan y cómo se utilizan a través del panel de configuración de privacidad en el sistema. Transparencia: Microsoft proporciona herramientas y opciones para revisar y gestionar los datos que se recopilan, incluyendo la opción de eliminar ciertos datos o desactivar la recopilación de datos específicos. Compartición de Datos:

Con Socios de Microsoft: En algunos casos, los datos pueden ser compartidos con socios de Microsoft para proporcionar servicios y soporte. Sin embargo, Microsoft afirma que no vende datos personales a terceros. Cumplimiento Legal: Los datos también pueden ser compartidos si es necesario para cumplir con requisitos legales o proteger los derechos de Microsoft y sus usuarios. Protección de Datos:

Seguridad de Datos: Microsoft emplea medidas de seguridad para proteger los datos personales recopilados y para evitar accesos no autorizados. Acceso y Corrección: Los usuarios pueden acceder a sus datos personales y solicitar correcciones o eliminación a través de las configuraciones de privacidad de Microsoft. Documentación en Windows 10:

Abre el "Inicio". Ve a "Configuración" (ícono de engranaje). Selecciona "Privacidad". Aquí encontrarás opciones para gestionar tu privacidad y enlaces a la política de privacidad completa. Consideraciones Adicionales Actualizaciones de Política: Microsoft puede actualizar su política de privacidad y las prácticas de recopilación de datos con el tiempo. Es recomendable revisar periódicamente la política para estar al tanto de cualquier cambio. Consentimiento del Usuario: Durante la instalación de Windows 10 y en las actualizaciones, se te pedirá que aceptes los términos y condiciones, incluyendo la política de privacidad. Si necesitas información específica o tienes inquietudes sobre la privacidad en Windows 10, te recomiendo consultar directamente la política en el sitio web de Microsoft o contactar con su soporte técnico.
